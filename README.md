# 🎭 Playwright Learning Journey

This repository contains my hands-on practice while learning Playwright with TypeScript. It covers everything from Playwright fundamentals to advanced test architecture and best practices.

## 🚀 Topics Covered

### Playwright Setup
- Playwright installation and configuration
- Browser projects and configuration management
- Playwright Test Runner
- VS Code extension
- UI Mode
- Package management using npm/pnpm

### Test Automation
- Creating automated tests
- Code generation (Codegen)
- Locator strategies
- Assertions
- Test organization and structure
- Authentication using stored cookies
- Visual testing
- API testing
- Best practices for reliable automation

### Test Maintenance
- Screenshots and video recording
- HTML reports
- Trace Viewer
- Scaling Playwright test suites
- Debugging failed tests

### Advanced Playwright Concepts
- Page Object Model (POM)
- Test data management
- Data Factory pattern
- Helper functions
- Custom assertions
- TypeScript configuration
- Reusable test abstractions

### Fixtures
- Custom fixtures
- Advanced fixtures
- Dependency injection using fixtures
- Shared test setup and teardown

## 🛠 Technologies Used

- Playwright
- TypeScript
- Node.js
- VS Code
- Git & GitHub

## 📚 Learning Outcomes

Through this repository I learned how to:

- Build maintainable end-to-end test automation frameworks.
- Apply the Page Object Model for reusable test code.
- Manage test data efficiently using Data Factory patterns.
- Create reusable helper functions and custom assertions.
- Use Playwright fixtures to simplify test setup.
- Perform UI, API, authentication, and visual testing.
- Debug failures using Trace Viewer and HTML reports.
- Structure Playwright projects following industry best practices.
